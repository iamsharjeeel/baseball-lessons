# Legacy Vite Build (v1 — PR #1)

This folder preserves the original Vite + React + Tailwind v4 scaffold from PR #1. It is **not** the active application.

The v2 rebuild uses **Next.js 15 App Router + GSAP** at the repo root. See `AGENT_INSTRUCTIONS.md` and `HANDOVER.md` for the current stack.

To run this legacy build for reference:

```bash
cd legacy-vite-build
npm install
npm run dev
```
